## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Python 3+
 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/HNagouda/House-Prices-EDA.git
   ```
2. Install required packages
   ```sh
   pip install -r requirements.txt
   ```
